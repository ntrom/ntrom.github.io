Getting into programming huh? Or, seasoned expert? Alright cool, well either way, I'd love to show ya a list of tools, technologies, languages, and more, that have made my life as a programmer what it is today.

Compilers and Programming Languages

The underlying systems that allows programmers to operate on a logical level (vs. a physical/machine-driven level) are programming languages, and their corresponding tools called compilers, which translate the logical declaration of operations into a sequence of machine instructions that perform the underlying low-level mathematical operations.

C
Commonly used for: Operating systems, embedded systems, games, drivers, other software with requirements for high performance
C++
Commonly used for: Games, drivers, high performance web application servers
C#
Commonly used for: Web application servers, Windows user interface applications, gameplay scripting
Java
Commonly used for: Web application servers, Cross-platform user interface applications
Python
Commonly used for: Web application servers, various scripting purposes

Integrated Development Environments (IDEs)

IDEs are the tool a programmer lives in every day. It contains compilers, text editors with syntax coloring, and many other features that facilitate rapid development. Of course, you can always code without an IDE at the end of the day (compiling on the command line), but using an IDE is the most effective and efficient way to stay productive.
